## 1. Вход в систему

1. Откройте сайт конкурса
2. Нажмите кнопку **«Войти»** в шапке сайта
3. Введите email и пароль администратора
4. Нажмите **«Войти»**

**Учётные данные администратора по умолчанию:**
- Email: `admin@nasledniki-pobedy.ru`
- Пароль: `admin123`

> ⚠️ **Важно:** Рекомендуется сменить пароль после первого входа.

## 2. Обзор админ-панели

После входа вы попадёте на главную страницу админ-панели с:

### Карточки статистики
- **Участников** — количество зарегистрированных школьников
- **Экспертов** — количество созданных экспертов
- **Работ** — общее количество поданных работ
- **Средний балл** — средняя оценка по всем работам

### Графики
- **Работы по номинациям** — круговая диаграмма (ВОВ / СВО)
- **Работы по статусам** — столбчатая диаграмма
- **Топ школ** — рейтинг школ по количеству работ

### Быстрые действия
- Пользователи
- Работы
- Создать эксперта
- Настройки

### Предупреждения
Если есть работы без назначенного эксперта — появится предупреждение с кнопкой «Распределить».

## 3. Управление пользователями

**Путь:** Админ-панель → Пользователи

### Фильтрация
- **Все** — все пользователи системы
- **Участники** — только школьники
- **Эксперты** — только эксперты
- **Админы** — только администраторы

### Действия с пользователем

| Действие | Описание |
|----------|----------|
| **Заблокировать** | Пользователь не сможет войти в систему |
| **Разблокировать** | Снятие блокировки |
| **Сбросить пароль** | Отправка email со ссылкой для сброса пароля |

---

## 4. Создание эксперта

**Путь:** Админ-панель → Создать эксперта

1. Заполните форму:
   - **ФИО** — полное имя эксперта
   - **Email** — электронная почта (для входа)
   - **Пароль** — нажмите «Сгенерировать» или введите вручную

2. Нажмите **«Создать эксперта»**

3. Передайте эксперту:
   - Email
   - Сгенерированный пароль
   - Ссылку на сайт

> Эксперт сможет войти и увидеть только назначенные ему работы.

## 5. Управление работами

**Путь:** Админ-панель → Работы

### Фильтры
- **Все** — все работы
- **Без эксперта** — работы, ожидающие назначения
- **На проверке** — работы, назначенные экспертам
- **Оценено** — работы с выставленной оценкой

### Ручное назначение эксперта

1. Выберите эксперта из выпадающего списка
2. Отметьте галочками работы для назначения
3. Нажмите **«Назначить»**

### Информация о работе
- Название
- Номинация (ВОВ/СВО)
- Тип (сочинение/рисунок)
- Автор (ФИО, школа, класс)
- Статус
- Назначенный эксперт
- Оценка (если есть)

### Действия
- **Скачать** — загрузить файл работы
- **Снять эксперта** — вернуть работу в статус «На модерации»

---

## 6. Автораспределение работ

**Путь:** Админ-панель → Работы → кнопка «Автораспределение»

Функция автоматически распределяет все работы без эксперта между доступными экспертами.

**Алгоритм:**
1. Берутся все работы со статусом «На модерации» без эксперта
2. Берутся все активные (не заблокированные) эксперты
3. Работы распределяются равномерно с учётом текущей нагрузки
4. Каждому эксперту отправляется email-уведомление

---

## 7. Настройки системы

**Путь:** Админ-панель → Настройки

### Дедлайн подачи работ
- Установите дату и время окончания приёма работ
- После дедлайна участники не смогут подавать новые работы

### Максимальный балл
- Настройте шкалу оценки (например, от 1 до 10)
- Эксперты будут выставлять оценки в рамках этой шкалы

---

## 8. Экспорт данных

**Путь:** Админ-панель → кнопки экспорта в шапке

### Экспорт работ
Выгружает Excel-файл со всеми работами:
- ФИО участника
- Школа, класс
- Название работы
- Номинация, тип
- Статус
- Эксперт
- Оценка

### Экспорт результатов
Выгружает Excel-файл только с оценёнными работами для подведения итогов.

## 9. Работа эксперта

### Вход
1. Перейти на сайт
2. Нажать «Войти»
3. Ввести email и пароль, полученные от администратора

### Личный кабинет эксперта
- Список назначенных работ
- Статус каждой работы (ожидает оценки / оценено)

### Оценка работы
1. Нажать на работу в списке
2. Скачать и изучить файл
3. Выставить оценку (от 1 до макс. балла)
4. Написать комментарий (опционально)
5. Нажать «Сохранить оценку»

> После оценки участник получит email-уведомление.

---

## 10. Работа участника

### Регистрация
1. Нажать «Зарегистрироваться»
2. Заполнить форму (ФИО, email, школа, класс)
3. Принять политику конфиденциальности
4. Нажать «Зарегистрироваться»

### Подача работы
1. Войти в личный кабинет
2. Нажать «Подать работу»
3. Заполнить:
   - Название работы
   - Номинация (ВОВ или СВО)
   - Тип (сочинение или рисунок)
   - Загрузить файл (PDF, DOC, DOCX, JPG, PNG)
4. Нажать «Отправить»

### Ограничения
- Одна работа на каждую номинацию (максимум 2 работы)
- Размер файла — до 15 МБ
- Подача возможна до дедлайна

### Редактирование работы
Пока работа в статусе «На модерации», участник может:
- Изменить название
- Заменить файл
- Удалить работу
